Типи даних в програмуванні

Типи даних визначають, які значення може приймати змінна і які операції над цими значеннями можна виконувати. Розуміння типів даних допомагає ефективно програмувати та уникати помилок.

Основні типи даних з прикладами

Тип Опис Приклад значення Приклад коду (Python)
Цілі числа (Integer) Зберігають ціле число, без дробової частини 5, -10, 0 
x = 10
y = -3
print(x + y)
# Виведе 7
Дробові числа (Float) Зберігають числа з десятковою частиною 3.14, -0.5, 2.0 
a = 3.5
b = 2.0
print(a * b)
# Виведе 7.0
Рядки (String) Зберігають текстову інформацію "Привіт", "123" 
name = "Кий"
print("Привіт, " + name)
# Виведе "Привіт, Кий"
Логічний тип (Boolean) Має лише два значення: правда або неправда 
False, True
0     1
                            
is_raining = True
if is_raining:
    print("Візьми парасольку!")

Підсумок: Знання типів даних і прикладів їх використання дозволяє правильно обробляти інформацію в програмах і уникати помилок при роботі зі змінними.
